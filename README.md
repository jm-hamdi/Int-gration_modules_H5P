# Projet H5P - Test Technique

##  Description
Application web permettant d’intégrer et d’afficher des modules **H5P interactifs** directement dans la plateforme.  
Le projet vise à tester l’intégration de contenus interactifs, la structure du code et le déploiement sur un hébergement en ligne.

##  Fonctionnalités
- Intégration de modules H5P (quiz, vidéos interactives, présentations, etc.)
- Affichage dynamique des contenus H5P
- Déploiement en ligne pour faciliter les tests
